Check Weather, Gate And Flight Information

Note: Gate Queries Are Currently Only Available for United Flights in Newark.

Try the following Examples:

C101: Lists all scheduled departures from the requested gate. You can just search for `X` in this case. Only applicable for Newark at the moment.

492: Returns D-ATIS, METAR, TAF, and ground delay program, if any, at both departure and arrival airports for the given flight number - in this case 'UA492'. Currently Only available for United flights.

KEWR: Returns the latest D-ATIS(if available), METAR, TAF and ground delay program for the given airport.

Empty search returns information on all gates for United flights in KEWR.
